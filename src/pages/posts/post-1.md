## Pico8 Dev Log - 1 

I thought it was about time to start a devblog for the Pico8 game that I'm working on. What started as a little side project is quickly becoming one of my favorite things to work on in some of my downtime...and it's fun!  

### What I have so far

1. A "baseball-esque" game based on Blitzball
2. Written in a paired down version of Lua
3. Has its own testing suite that I built from the ground up

I wish I would have documented more as I was building it, but I'm going to create pages for all the big lessons I've learned / what I'm proud of in some separate posts as a kinda post mortem.
